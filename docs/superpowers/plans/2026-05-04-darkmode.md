# Dark Mode (HR Only) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a dark mode toggle to the HR desktop interface that switches the entire HR layout to a teal-tinted dark palette, persisted to localStorage, with no effect on employee screens.

**Architecture:** A `.dark {}` CSS block in `globals.css` redefines all `--color-*` custom properties. The `dark` class is applied to the HR layout wrapper div (not `<html>`), so employee screens are unaffected. A `useDarkMode` hook manages state + localStorage. A sun/moon icon button in the HR topbar triggers the toggle.

**Tech Stack:** Next.js 14, Tailwind CSS (`darkMode: ["class"]`), CSS custom properties, React hooks, lucide-react icons, localStorage

## Task 2: Create `useDarkMode` hook

**Files:**
- Create: `lib/hooks/use-dark-mode.ts`

- [ ] **Step 1: Create the hooks directory and file**

```bash
mkdir -p lib/hooks
```

Create `lib/hooks/use-dark-mode.ts` with this content:

```ts
"use client";

import { useEffect, useState } from "react";

const STORAGE_KEY = "payday-dark-mode";

export function useDarkMode(): [boolean, () => void] {
  const [isDark, setIsDark] = useState<boolean>(() => {
    if (typeof window === "undefined") return false;
    return localStorage.getItem(STORAGE_KEY) === "true";
  });

  const toggle = () => {
    setIsDark((prev) => {
      const next = !prev;
      localStorage.setItem(STORAGE_KEY, String(next));
      return next;
    });
  };

  return [isDark, toggle];
}
```

## Task 3: Create `DarkModeToggle` component

**Files:**
- Create: `components/ui/dark-mode-toggle.tsx`

- [ ] **Step 1: Create the component file**

Create `components/ui/dark-mode-toggle.tsx`:

```tsx
"use client";

import { Moon, Sun } from "lucide-react";
import { useDarkMode } from "@/lib/hooks/use-dark-mode";

export function DarkModeToggle() {
  const [isDark, toggle] = useDarkMode();

  return (
    <button
      type="button"
      onClick={toggle}
      className="flex h-8 w-8 items-center justify-center rounded-md text-slate-500 transition hover:bg-bg-secondary hover:text-primary-dark focus:outline-none focus:ring-2 focus:ring-primary/30"
      aria-label="เปิด/ปิดโหมดสีเข้ม"
      aria-pressed={isDark}
    >
      {isDark ? (
        <Sun className="h-5 w-5" strokeWidth={1.8} aria-hidden />
      ) : (
        <Moon className="h-5 w-5" strokeWidth={1.8} aria-hidden />
      )}
    </button>
  );
}
```

- [ ] **Step 1: Rewrite `hr-layout.tsx` to be a client component that applies the dark class**

```tsx
"use client";

import type { ReactNode } from "react";
import { HRSidebar } from "./hr-sidebar";
import { HRTopbar } from "./hr-topbar";
import { useDarkMode } from "@/lib/hooks/use-dark-mode";
import { cn } from "@/lib/utils";

interface HRLayoutShellProps {
  children: ReactNode;
}

export function HRLayoutShell({ children }: HRLayoutShellProps) {
  const [isDark] = useDarkMode();

  return (
    <div className={cn("flex min-h-screen bg-bg-page text-text-primary", isDark && "dark")}>
      <HRSidebar />
      <div className="flex min-w-0 flex-1 flex-col">
        <HRTopbar />
        <main className="min-h-0 flex-1 overflow-auto px-5 py-5">
          {children}
        </main>
      </div>
    </div>
  );
}
```
